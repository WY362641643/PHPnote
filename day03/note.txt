位运算符
  &(与) |(或)  ~(非)

按位左移 <<

错误抑制符 @
    在一个表达式出现错误的时候, 可以将错误"隐藏起来"

运算符的优先级
    1.时刻意识到,运算符有优先级
    2.小括号为最高优先级
    3.先乘除(取余)再加减
    4.算数运算 > 比较运算 > 逻辑运算
    5.复制运算符通常都是最后(优先级最低)

条件运算符
    表达式1 ? 表达式2 : 表达式3
    含义:
        如果表达式1的运算结果为 True 则执行表达式2, 否则为表达式3


流程控制:
    if 语句
     ...

    switch 语句
        switch(一个变量数据或表达式结果$v1){
            case 值1:        // 如果$v1的值, 等于这个 '值1', 就执行本分支
                分支语句1;
                break;
            case 值2:
                分支语句2;
                break;
            ...
            default:
                默认分支;   // 前面都不满足的时候, 执行这里
        }

    while 循环语句
        while( 条件判断 ){
            循环语句;
            // 改变循环变量
            条件--(条件++);
        }

    do while 循环语句
        do{
            循环语句
        }while(条件判断);
            先执行一次循环语句, 然后在进行条件判断:
            如果成立,则继续回去执行循环体, 而后再次进行条件判断, 如此反复;
            如果不成立, 这退出循环,执行后续语句

    for 循环语法:
        for (循环变量初始化1; 循环条件判断2; 循环变量改变3){
            执行循环语句块
        }
        列:
            for($i=1; $i<=100; $i++){
                执行循环语句块
            }
